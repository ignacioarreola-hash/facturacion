# Conector de Facturación CFDI 4.0 — Colossu

Roadmap interactivo y prototipos. Abre `index.html` para ver el flujo clickeable.

## Publicar en GitHub Pages (importante: subir los archivos SUELTOS)

1. Crea un repo nuevo en GitHub.
2. Sube TODOS estos archivos a la RAÍZ del repo (no dentro de una carpeta).
   El `index.html` debe quedar directo en la raíz, junto a los demás.
3. Settings → Pages → Source: Deploy from a branch → main / (root) → Save.
4. Espera 1–2 min. Estará en https://<usuario>.github.io/<repo>/
